[tasklist] - Show running processes

[windowsdiskcheck] or [wdc] - Run chkdsk on system drive
[systemfilecheck] or [sfc] - Run system file checker

  blue, green, aqua, red, purple, yellow, white, gray
  light blue, light green, light aqua, light red
  light purple, light yellow, bright white, default

[calc] or [calculator] - Open calculator
[coinflip] - Flip a coin (50/50)
[wp] or [wordpad] - Open WordPad
[notepad] or [np] - Open Notepad

[msg] or [message] - Send message popup
[com] - View this command list
[report] - Send feedback via form
[upd] or [update] - Check for updates
[checkupd] - Check version silently
[info] or [i] - Information menu
  ver, license, disclaimer, return
[solar] or [s] - GitHub files menu
  needed, other, read, all

  Device Manager, Disk Management, Services, Event Viewer
  System Info, Control Panel, Programs, Network Connections

  systeminfo, taskkill, DISM, driverquery, powercfg
